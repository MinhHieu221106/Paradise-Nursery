# Paradise Nursery

Paradise Nursery is a React-based e-commerce web application for an online 
houseplant store. The app allows customers to browse a curated selection 
of houseplants organized by category, add items to a shopping cart, and 
manage their cart quantities before checkout.

## Tech Stack
- React
- Redux Toolkit (state management for the shopping cart)
- React Router (page navigation)
- CSS

## Features
- Landing page introducing the company
- Product listing page grouped by plant category
- Shopping cart with add/remove/update quantity functionality
- Responsive navigation bar across all pages

## Project Structure
- `App.jsx` – Main landing page
- `AboutUs.jsx` – Company information
- `ProductList.jsx` – Product listing page
- `CartItem.jsx` – Shopping cart page
- `CartSlice.jsx` – Redux slice for cart state management
